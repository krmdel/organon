# Statistical Analysis Report Template

Template used by `data_ops.generate_report()` for structured output. Also serves as a reference for manually formatting results.

---

## Template

```markdown
# Statistical Analysis Report

**Date:** {YYYY-MM-DD}
**Dataset:** {filename} ({rows} rows x {cols} columns)
**Analysis:** {test_name}

## Test Selected

{test_name} was selected because {rationale}.

## Assumptions Check

| Assumption | Test | Statistic | p-value | Pass/Fail | Note |
|-----------|------|-----------|---------|-----------|------|
| Normality (Group A) | Shapiro-Wilk | {W} | {p} | {pass/fail} | {interpretation} |
| Normality (Group B) | Shapiro-Wilk | {W} | {p} | {pass/fail} | {interpretation} |
| Homogeneity of variance | Levene's | {F} | {p} | {pass/fail} | {interpretation} |

{If any assumptions failed:}
**Warning:** {assumption} was violated (p = {p}). Consider using {alternative} instead.
The scientist may choose to proceed with the current test or switch to the alternative.

## Results

| Metric | Value |
|--------|-------|
| Test statistic | {value} |
| p-value | {value} |
| Effect size ({measure}) | {value} ({interpretation: small/medium/large}) |
| 95% Confidence interval | [{lower}, {upper}] |

## Interpretation

{Plain-English summary. Examples:}

For t-test: "The difference between Group A (M = X, SD = Y) and Group B (M = X, SD = Y) was [statistically significant / not statistically significant], t(df) = X, p = Y. The effect size (Cohen's d = Z) suggests a [small/medium/large] practical difference."

For ANOVA: "There was a [significant / non-significant] difference among groups, F(df_between, df_within) = X, p = Y, eta-squared = Z."

For correlation: "There was a [weak/moderate/strong] [positive/negative] correlation between X and Y, r = Z, p = W."

For chi-square: "There was a [significant / non-significant] association between X and Y, chi2(df) = Z, p = W, Cramer's V = V."

## Recommendation

{Context-dependent suggestions:}

- If assumptions violated: "Consider using {alternative test} which does not require {violated assumption}."
- If significant with ANOVA: "Run post-hoc pairwise comparisons (Tukey HSD) to identify which groups differ."
- If not significant: "The sample may lack statistical power. Consider increasing sample size or using a more sensitive design."
- If effect size is small but significant: "While statistically significant, the effect size is small -- consider practical significance."

---
*Generated by sci-data-analysis | Logged to reproducibility ledger*
```

---

## Field Mappings from data_ops.generate_report()

The `generate_report()` function in `data_ops.py` automatically populates:

| Template field | Source in test_results dict |
|---------------|---------------------------|
| test_name | `test_results["test_name"]` |
| Assumptions table | `test_results["assumptions"]` (nested dicts with statistic, p_value, passed) |
| Test statistic | `test_results["statistic"]` |
| p-value | `test_results["p_value"]` |
| Effect size | `test_results["effect_size"]` + `test_results["effect_size_label"]` |
| 95% CI | `test_results["ci_95"]` (tuple) |
| Significant | `test_results["significant"]` (bool) |
| Alternative | `test_results["alternative_suggested"]` (if assumptions failed) |
| Post-hoc | `test_results["post_hoc_guidance"]` (for ANOVA) |

The report header (Date, Dataset) should be added by the skill when saving the file.
